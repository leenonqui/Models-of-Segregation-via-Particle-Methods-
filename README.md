# Schelling's Model of Segregation
### Homework 1 — Particle Methods, Spring 2026
**Author:** Onorato Lee Johnson

A 2D particle simulation based on Thomas C. Schelling's model of residential segregation.

## Reference
T. C. Schelling, "Models of Segregation," *The American Economic Review*, vol. 59, no. 2, pp. 488–493, 1969. https://www.jstor.org/stable/1823701

## Requirements
- Python 3.10+
- numpy
- matplotlib

Install with:
```bash
pip install numpy matplotlib
```

## Usage
Open `notebook.ipynb` to run simulations and view results.
